## Usage Guidelines

### In Documentation

- Use the ™ symbol on the **first prominent mention** of each trademark in a document or page.
- Subsequent mentions within the same document may omit the ™ symbol.
- Always capitalize trademarks as shown above. Do not use them in lowercase or as generic terms.

### Examples

**Correct:**
- "Tri-Lane Protocol™ is an original Vastar architecture."
- "VIL™ is a process-oriented language and framework."
- "Data flows through the Tri-Lane Protocol without head-of-line blocking." (subsequent mention, ™ not required)

**Incorrect:**
- "tri-lane protocol" (lowercase)
- "The trilane" (altered form)
- "a VIL" (used as generic noun)

### In Source Code

Trademark symbols are not required in source code, configuration files, or CLI output. They are required only in documentation, marketing materials, and website content.
