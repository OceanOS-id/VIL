**VIL** (Vastar Intermediate Language) is a trademark of **OceanOS** and **Vastar AI**.

The VIL name, logo, and associated branding are protected to ensure consistent quality and identity for the community.

## Permitted Use

You **may** use the VIL name to:

- Refer to the VIL project, framework, or ecosystem in documentation, articles, talks, and tutorials
- State compatibility (e.g., "compatible with VIL", "built for VIL", "works with VIL pipelines")
- Name community plugins using the pattern `vil-<name>` or `vil_<name>` (e.g., `vil-my-plugin`)
- Fork and modify the source code under the terms of the MIT or Apache-2.0 license

## Restricted Use

You **may not** use the VIL name to:

- Imply official endorsement or affiliation without written permission
- Name a competing or unrelated product "VIL" or any confusingly similar name
- Use the VIL name in a company name, commercial product name, or domain name without written permission
- Distribute modified versions under the VIL name without clearly stating the modifications

## Modified Versions

If you distribute a modified version of VIL, you must:

1. Clearly indicate that it is a modified version (e.g., "Based on VIL" or "Fork of VIL")
2. Not use the VIL name as the primary product name
3. Include a prominent notice describing the modifications

## Community Plugins

Community-contributed plugins and extensions **may** use the `vil-` or `vil_` prefix (e.g., `vil_my_storage`, `vil-xml-parser`). This does not imply official endorsement. The [vil_plugin_sdk](https://github.com/OceanOS-id/VIL) provides the stable interface for building plugins.
